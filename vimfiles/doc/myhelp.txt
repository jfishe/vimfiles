2. Visual Mode                                   *myhelp-visual*

J   Join the highlighted lines, removing unnecessary whitespace.
va' Select a single-quoted string (also works with " and `)
vi' Select the contents of a single-quoted string
vab Select a "block" (works differently in different langs)
vat Select a <tag> and its contents (use i instead of a for just contents)
date` Insert today's date.

4. Ex                                            *myhelp-ex*

Command                     Meaning ~
:Vterm                      Default vertical terminal. Append bash to specify.
:Vposh                      PowerShell vertical terminal
:set relativenumber         |'relativenumber'| |'rnu'|
:s/ <\p\{-}@\p\{-}>//cgi    Remove email addresses.
:%s/\%VSEARCH/REPLACE/g     Replace all occurrences in selection but not
                            line. \%V limits the search.
:%s/\%V <\p\{-}@\p\{-}>//g  Remove all email address in line or selection.
:%s/[{;}]/&\r/g|norm! =gg   Format css from one-line to multi-line.

                                                *myhelp-ex-pack*
Vim does not generate help tags for pack/foo/opt/doc because opt/doc is not in
the runtime path, nor does include existing tags that are not in the runtime
path.

Command                     Meaning ~
:packadd! foo               Add opt/foo to runtime path.
                            :help foo will work if tags exist.
:Myhelptags                 Create/update helptags under opt.
